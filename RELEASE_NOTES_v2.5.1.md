# ShowCall v2.5.1 - Release Notes

**Release Date:** May 12, 2026

## 🎯 What's New

### Preset Deck Customization ✨
Your preset popout deck is now fully customizable! Click the hamburger menu (☰) in the top-right corner to:

- **Toggle Cue Go Button** - Large green button to execute next cue (enabled by default)
- **Toggle Cue Status** - See your progress (e.g., "Ready: Cue 3 of 12")
- **Toggle Quick Cues** - Ready for future quick action buttons

Settings are saved automatically and persist across sessions!

### Cue Stack Control 🎬
- **GO Button in Deck** - Large, prominent green button for executing cues
- **Cue API** - New backend endpoints for cue management (`/api/cuestack`, `/api/cuestack/execute`)
- **Visual Feedback** - Status displays and smooth interactions

### Build & Release Improvements 🔧
- Fixed macOS notarization workflow (back to env var approach)
- Added credential validation in CI/CD pipeline
- Verbose DEBUG logging for troubleshooting
